# 📝 Resumaker-feature - 简历编辑器

本项目基于 [resumaker](https://github.com/WhiteP1ay/resumaker) 扩展而来，在原项目的基础上新增了一些致力于用户体验优化的特性和功能。

纯前端项目，没有登录注册、信息收集，也没有收费项目。

## 🚀 快速开始

### 环境要求
- Node.js >= 18.0.0
- npm 或 yarn 或 pnpm

### 安装依赖

```bash
# 使用 npm
npm install
```

### 开发运行

```bash
# 启动开发服务器
npm run dev
```

### 构建部署

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```
